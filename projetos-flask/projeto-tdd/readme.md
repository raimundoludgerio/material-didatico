# XP Project Base

Projeto base para prática de Extreme Programming (XP).

## Objetivo
Aplicar TDD, pair programming, pequenas entregas e refatoração contínua.

## Regras XP do Projeto
- Nenhuma funcionalidade sem teste
- Código simples primeiro
- Refatorar sempre que necessário
- Commits pequenos e frequentes
- Troca de pares a cada iteração
