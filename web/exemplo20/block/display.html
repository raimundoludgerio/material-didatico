Organizando os elementos em block com Display:block

No CSS, a propriedade display define como um elemento deve ser exibido na página. Um dos valores mais comuns dessa propriedade é o block. Quando você aplica display: block a um elemento, está dizendo ao navegador que ele deve se comportar como um elemento de bloco. Isso significa que:

  • O elemento ocupará toda a largura disponível do seu contêiner pai, por padrão.
  • Ele sempre começa em uma nova linha, empurrando os elementos seguintes para baixo.
  • É possível definir largura (width), altura (height), margens (margin), preenchimentos (padding), entre outras propriedades de layout..
            
    Exemplos de elementos com display: block por padrão:
    <h1>Div</h1>
    <h1>h1</h1>
    <h2>h2</h2>
    <section>section</section>
    <article>article</article>
    <p>p</p>
